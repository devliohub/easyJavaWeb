# easyJavaWeb
### 简易的java web框架， 自行手搓进行扩展， 学习研究使用
#####  _A simple java web framework, extensible by hand, for learning and research_

***

> 愿景：
>    中小企业，解决在线100W并发，引导形成一个可读性、二次修改方面友好的框架，以致于各种类型的程序员灵活开箱使用。
 + 中间件客户端
   + mysql
   + redis
   + 消息队列
     + kafak
     + rocketMQ
     + rabbitMQ
   + Memcached
   + MongoDB
   + Elasticsearch
+ 工程能力
   + docker
   + 微服务
   + 操作系统
     + 网络
     + 内存
     + CPU
     + 磁盘
     + 文件系统
   + 分布式
 
***
